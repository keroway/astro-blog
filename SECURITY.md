# Security Policy

## Supported Versions

このリポジトリは個人のポートフォリオ・ブログサイト（keroway.com）のソースコードです。
最新の `main` ブランチのみをサポートしています。

## Reporting a Vulnerability

脆弱性を発見した場合は、**public な Issue には投稿しないでください**。

代わりに GitHub の [Private vulnerability reporting](https://github.com/keroway/astro-blog/security/advisories/new) を利用して報告してください。

報告いただいた内容は確認次第対応します（個人開発のため、対応時期は状況によります）。

## Scope

- `src/` 以下のソースコード
- `scripts/` 以下の運用スクリプト
- 依存パッケージの脆弱性（Dependabot alert が未対応のもの）

ブログ記事コンテンツ（`src/content/`）のテキスト内容はスコープ外です。
